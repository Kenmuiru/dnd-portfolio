The GitHub page is open at: https://github.com/new

## What to Do:

**Fill in the form on the GitHub page:**

1. **Repository name**: `dnd-portfolio`
2. **Description**: `DnD Couture Portfolio Website with Node.js backend`
3. **Public or Private**: Choose **Public** (recommended)
4. **CRITICAL**: Do NOT check any of these boxes:
   - ❌ Add a README file
   - ❌ Add .gitignore
   - ❌ Choose a license

5. Click the green **"Create repository"** button

## After Creating:

GitHub will show you a page with your repository URL:
```

```

**I need your GitHub username from that URL!**

For example, if the URL is:
```
https://github.com/kenmuiru/dnd-portfolio.git
```

Then your username is: `kenmuiru`
